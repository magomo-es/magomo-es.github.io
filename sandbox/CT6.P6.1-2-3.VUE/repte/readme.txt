Minirepte amb Visual Studio Code
--------------------------------------------------------------------------------------

- Fes un minirepte propi amb Visual Studio Code fent ús de:
* directives Vue (https://es.vuejs.org/v2/api/#Directivas): v-text, v-model, v-if, v-for, v-bind, v-on, Computed properties, props, emits, Class & Style Binding
* Elements pare i elements fill amb comunicació bidireccional.
* Formularis (amb validació).

- Us recordo que heu de crear el vostre portfoli i això us brinda una oportunitat per aplicar els vostres coneixements en un projecte real, 
aprofitant així el vostre temps d’aprenentatge. Tenint això en ment podeu fer coses com: “crear un formulari que serveixi per donar d’alta 
les vostres competències tècniques i el nivell de coneixement actual. Aquestes competències es mostren en una llista a la vegada que a la 
capçalera hi ha un comptador amb el nombre de competències tècniques afegides. Segons el nivell de coneixement es mostra algun indicador 
representatiu. Si entrem en mode edició podem cambiar el nivell”

- Aquesta és únicament una proposta, segur que hi ha idees molt diferents i més adequades a la teva idea de portfoli. Facis el que facis 
recorda que ens ha de permetre aprendre i aprofitar de manera òptima el temps de feina, així doncs:  Feel free!  

- Adjunta a la tasca de l’aula virtual un arxiu comprimit amb el teu codi.
