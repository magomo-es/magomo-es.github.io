Proposta d’estructura de carpetes base per les nostres webApp’s:

📁  webAppEnBlanc //Carpeta contenedora de la webApp 
📄 readme.txt //Presentació de l’aplicació web
📄 index.html //Página principal de la webApp amb DOM, layout i componentes de la webApp

📁 js    // Carpeta de funcionalitats
📄 config.js // Configuració de la webApp
📄 events.js // Eventss de la webApp
📄 tools.js  // Arxiu d’eines pròpies que es poden aprofitar en altres WebApp’s
📄 nomWebApp.js // Arxiu amb les funcionalitats pròpies de la webApp

📁 lang // Carpeta amb recursos multiidioma (vectors claur/valor)
📄 CAT.js // Literals en català
📄 ENG.js // Literals en anglès
📄 ESP.js // Literals en castellà

📁 media // Carpeta amb recursos multimedia genèrics

📁 users // Carpeta amb recursos multimedia generats pels usuaris

📁 webApp // Carpeta amb recursos multimedia propis de la webApp, per exemple les icones o imatges de 
la interfície. 

📁 style // Carpeta amb l’apariència de la webApp
📄 nomWebApp.css // Full d’estil de la webApp

📁 data  // Carpeta amb dades de la webApp (JSON’s, XML’s, Variables globals amb vectors clau valor...) 
pot servir per simular els contenidors de dades persistents externs (dades en bases de dades)

📁 tercers  // Plugins, llibreries, arxius, recursos… de 3ers (si és que ens ho permeten i no ens ho 
contraindiquen les instruccions d’ús d’aquests recursos)

📁 UI  // Recursos relacionats amb la interfície d’usuari pròpia o de tercers (Bootstrap, material design…
